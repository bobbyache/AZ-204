What does the Microsoft identity platform help you do?|The Microsoft identity platform helps you build applications your users and customers can sign in to using their Microsoft identities or social accounts, and provide authorized access to your own APIs or Microsoft APIs like Microsoft Graph.
What are the components of the Microsoft identity platform?|The components of the Microsoft identity platform include OAuth 2.0 and OpenID Connect standard-compliant authentication service, open-source libraries like Microsoft Authentication Libraries (MSAL) and support for other standards-compliant libraries, application management portal in the Azure portal, and application configuration API and PowerShell.
What are the different types of identities that can be authenticated using the Microsoft identity platform?|The different types of identities that can be authenticated using the Microsoft identity platform include work or school accounts provisioned through Azure Active Directory, personal Microsoft accounts like Skype, Xbox, and Outlook.com, and social or local accounts using Azure Active Directory B2C.
How can you programmatically configure your applications using the Microsoft identity platform?|You can programmatically configure your applications through the Microsoft Graph API and PowerShell to automate your DevOps tasks.
What modern innovations in the identity and security space does the Microsoft identity platform offer for developers?|The Microsoft identity platform offers integration of modern innovations like passwordless authentication, step-up authentication, and Conditional Access, allowing developers to take advantage of these functionalities without implementing them themselves.
